# Chord Sheets in ChordPro format

This repo contains a collection of chord sheets in the ChordPro format. Mostly Christian music.

## How to use
1. Clone this repo.
1. Download [ChordPro](https://github.com/ChordPro/chordpro/releases)
1. Enjoy!
